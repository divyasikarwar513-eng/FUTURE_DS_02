# FUTURE_DS_02 – Customer Retention & Churn Analysis

This project analyzes customer retention patterns and churn behavior using Microsoft Excel.

## Dashboard Features
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- KPI Metrics
- Customer Retention Insights

## Tools Used
- Microsoft Excel
- PivotTables
- Charts

## Key Insights
- Customers with month-to-month contracts exhibited the highest churn rate.
- Fiber optic users showed significantly higher churn behavior.
- Electronic check users demonstrated the highest customer churn.
- Long-term contract customers displayed stronger retention patterns.
- Automatic payment methods were associated with lower churn rates.
